# 🚌 IBuS — Sistema de Rutas de Buses · Sogamoso

Django 5.1 + PostgreSQL (Supabase) + Docker

---

## 🚀 Cómo correr el proyecto 
```bash (si es primera vez)
docker compose up --build
```

## si no es primera vez
docker compose up


Listo. Entra a → http://localhost:8000


### Parar el servidor:
docker compose down
```
